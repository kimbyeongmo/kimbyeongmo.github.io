## 게임 제작 배경
- 대다수의 사람들이 플레이해보았던 게임
- 어렸을 때부터 자주 접했던 아케이드 장르
- 게임 내 오브젝트의 간단한 함수 구현
- 복잡하지 않고 직관적인 UI
